Simulasi ATM Sederhana 
Proyek ini adalah simulasi ATM sederhana yang dibuat menggunakan Python dan GUI Tkinter.
Dengan penggunaan fundamental dasar:
- Variabel & Tipe Data
- Array (List & Dictionary)
- Pengkondisian (if, elif, else)
- Perulangan (while)
- Function
- GUI Tkinter

Pada aplikasi ini, pengguna dapat:
- Login menggunakan nomor rekening dan PIN
- Melihat saldo
- Menabung
- Menarik uang
- Keluar dari akun

Data rekening disimpan menggunakan dictionary
